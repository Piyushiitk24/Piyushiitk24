## 📝 Project Highlights  

- **🌐 VirtualLab Control Systems**  
  An interactive simulator for PID tuning and feedback loops. Ideal for students and hobbyists to visualise stability margins in real time.

- **🧠 Quizock**  
  A gamified quiz platform with adaptive question difficulty. Features live leaderboards and detailed analytics to keep learners engaged.

- **🔑 Offtoken**  
  A secure token-based authentication microservice built with Node.js and Redis. Simplifies single sign-on across distributed applications.

- **💡 SANKALP**  
  A community-driven idea incubation portal. Uses React & Firebase to let users pitch, collaborate and vote on tech innovations.

- **📚 EduLadder**  
  A step-by-step learning tracker for coding courses. Integrates with GitHub APIs to monitor progress and suggest personalised next steps.
